Berikut langkah-langkah untuk meng-host website kamu secara gratis di GitHub Pages:

  1. Buka situs github.com dan login akun kamu.
  2. Buat repository baru, misalnya: pemrogramanweb1.
  3. Upload file HTML, gambar, CSS, atau lainnya langsung ke repo tersebut.
  4. Pastikan file utamanya dengan nama yang sudah disesuaikan.
  5. Masuk ke tab Settings ‚Üí Pages.
  6. Di bagian "Build and deployment", pilih:
    • Branch: main
    • Folder: /root (bukan /docs)
  7. Klik Save, lalu tunggu beberapa saat.
  8. Link web kamu akan muncul, misalnya:
    https://username.github.io/pemrogramanweb1/
